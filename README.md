# Climas do Brasil - Weather App

## 📌 About the Project
**Climas do Brasil** is a mobile application developed in **Android Studio** that allows users to check real-time weather information for any city in Brazil. This project demonstrates the integration of external APIs to fetch live data and a robust system for internationalization.

## 🚀 Features
- **Real-time Weather Data:** Displays temperature (Celsius) and weather conditions via API.
- **Internationalization:** Built-in language switching system to change the app's language instantly.
- **Search Functionality:** Quickly find weather details for any Brazilian city.
- **Intuitive UI:** Minimalist and user-friendly layout designed for a great mobile experience.

## 🛠️ Technologies Used 
- **Android Studio**
- **Java**
- **XML**
- **Weather API Integration**
